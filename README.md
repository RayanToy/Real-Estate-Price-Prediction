# Real-Estate-Price-Prediction
Комплексный анализ данных и построение моделей машинного обучения для прогнозирования стоимости квартир. Проект включает в себя исследовательский анализ, предобработку данных с заполнением пропусков с помощью моделей, инжиниринг признаков, а также сравнение и оптимизацию регрессионных моделей, включая CatBoost, LightGBM, XGBoost и PyTorch.
